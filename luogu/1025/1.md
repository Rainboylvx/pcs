## 题目



### 题目描述

将整数$n$分成$k$份，且每份不能为空，任意两个方案不相同(不考虑顺序)。

例如：$n=7$，$k=3$，下面三种分法被认为是相同的。

$1,1,5$;  
$1,5,1$;   
$5,1,1$.

问有多少种不同的分法。




### 输入格式
$n,k$ ($6<n \le 200$，$2  \le k  \le  6$)




### 输出格式

$1$个整数，即不同的分法。




### 样例

|----|----|
|输入 #1  |输出 #1  |
|```  |```  \
|7 3  |4  \
|  |  \
|```  |```  |



### 说明/提示
四种分法为：  
$1,1,5$;  
$1,2,4$;  
$1,3,3$;  
$2,2,3$.



## 解析

可行性剪枝,上下界剪枝,每一层都有上界,下界.

保证不会出现重复,每一个元素都要比前一个元素大于等于.
