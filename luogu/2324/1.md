step+ h(x) > maxd-d+1

```c
int s = need_step();
if( s + d <= maxd+1){

```

为什么是`s+ d <= maxd+1`!!??

如果有$cnt$不同的位置,最少需要移动$cnt - 1$步,估价函数就写成$cnt-1+d <= maxd $=> $cnt+d<=maxd+1$
