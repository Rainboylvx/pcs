## 解析

`a*b`可以把a转成2进制表示

`(c*2^0 + c*2^1 + c*2^2 + c*2^3) *b`

-> `b*c*2^0 + b*c*2^1 + b*c*2^2 + b*c*2^3`

又因为

-> `b*2^i = b*2^{i-1} *2`

每一次的b都是上一个b的两倍

得出口诀

```
是1 就加
base倍增2
base 就是b
```

所以利用快速幂的思想,边加边取模
