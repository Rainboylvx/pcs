## 解析


这个题目的突破点在于,每一次只操作一个位置,一种类似于对顶堆的对顶栈作法

```
[ stack a ---> top] | [ top <---- stack b]
```

我们的栈前存下这个信息:`前k个元素的最大前缀和`,这个和`push,pop,getmin`这个题目一样,

我们用栈(或一个数组)来记录前k个元素的最大值,这显然是一个dp问题,

设`f[k]`表示,前k个元素的最大值

```
f[k] = max{f[k-1] , S[k-1] + a[k]}
```
